About

Project ini mengambil tema keuangan, 'Dompetku' hadir sebagai website yang mencatat aset seperti cash atau debit. Dengan menerapkan javascript dalam pencatatannya, 'Dompetku' akan menghasilkan informasi berupa tabel. Designnya dibuat lebih modern, hal ini dilakukan untuk melatih skill CSS. Perlu diingat, bahwa website ini hanya untuk pembelajaran pribadi semata. Dan data yang diinput tidak akan diambil atau masuk ke database, karena hasil tabel merupakan manipulasi javascript dan akan hilang ketika web direfresh.
